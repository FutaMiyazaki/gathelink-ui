## ディレクトリ構成

```
src/ ............. React に関するソースコード
├─ components .... 特定の feature を跨いで必要になるコンポーネント
├─ features ...... 機能毎の components, hooks, routes, types
├─ hooks ......... 各画面で共通して使用する Custom Hooks
├─ lib ........... 外部ライブラリ
├─ providers ..... Provider の設定
├─ routes ........ ルーティングの設定
├─ states ........ Recoil States
├─ types ......... 型定義
├─ utils ......... 定数や汎用関数
```
